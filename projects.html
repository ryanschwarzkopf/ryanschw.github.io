Flight Simulator Extension
This is my current project. Collaborating with Sandia National Laboratory and New Mexico State University students to develop Prepar3D toolkit extension for visualization of flight metrics using Agile project management. This toolkit will provide useful flight visualization features for Sandia engineers using Prepar3D. The toolkit will feature a 3d line following the aircraft's telemetric data, as well as a dashboard to display relevant flight metrics.
Predict Used Car Value
Worked with a team of New Mexico State University students to predict the price of a used car by applying various data-mining techniques. Individually contributed to data cleaning and trained a Multi-Layered Perceptron classifier with the Scikit-Learn framework. This task proved to be difficult to solve for a neural network however the support vector machine was able to predict the vehicles value reasonably well.
SARS-Cov-2
A search for most frequent K-Mers in Coronavirus revealed RNA synthesis starters. Mapped most common 9-mers in various Coronavirus strains onto a phylogenetic tree to infer ancestral sequences of Coronavirus.
C-MINUS+ to MIPS Assembly Compiler
Developed a minimal compiler for C-MINUS+ (a subset of C) to MIPS Assembly Language using Lex and Yacc tools. Written in C.
